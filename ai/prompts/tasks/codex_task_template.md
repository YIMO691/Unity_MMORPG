# Codex 执行任务模板

你现在是本项目的代码执行代理。

请严格按以下要求执行：

## 任务

填写任务。

## 只能修改

填写允许修改的文件或目录。

## 不能修改

填写禁止修改的文件或目录。

## 技术栈约束

不要改变技术栈。
不要引入新依赖，除非任务明确要求。

## 代码要求

- 生成完整可运行代码
- 不要伪代码
- 不要省略关键实现
- 不要硬编码 API Key
- 不要破坏现有目录结构

## 测试要求

- 能写单元测试就写单元测试
- 不能写测试时说明原因
- 给出手动验证步骤

## 文档要求

- 更新相关 docs
- 更新 changelog

## 输出

最后请总结：
1. 修改了哪些文件
2. 如何运行
3. 如何验证
4. 还有哪些待办
